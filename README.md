# 翠翔祭 2026 公式サイト

このリポジトリは 翠翔祭2026 の公式サイトのためのものです。

## 概要

本サイトは神奈川県立横浜翠嵐高等学校の生徒によって制作されており、翠翔祭の情報を提供することを目的としています。

## サイトについて

- 本サイトは非営利目的で運営されており、商業利用を一切行いません。

## 開発について

このプロジェクトは以下の技術を使用しています。

- **フロントエンド**: Next.js / React
- **デプロイ**: Vercel
- **スタイル**: Tailwind CSS / shadcn
